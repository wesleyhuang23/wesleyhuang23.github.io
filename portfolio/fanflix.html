Fanflix

Made for Movie Fans

Features

A Movie database website that allows user to search up to 900,000 movie titles. Searching is done in real time, making a api request on keypress. Users can browse movie detailed pages for every searchable movie, included are trailer, posters, backdrops and recommendations for the movie. Users can search for movie theaters in their area and get showtimes at that theater. When logging in clients will be able to create their own movie list and once added can write a review and give a score for the movie.

HTML/CSS | jQuery | AngularJS | NodeJS | Express | SQL | Gulp | OAuth | Heroku

Challenges

The Challenging part about this project was the back-end. Using OAuth in order to have a user ID be tracked with every action the user does so each user can create their own list of movies. Making sure that the back-end was properly connected. Having some properties be global while others be personal made it more overwhelming, but taking my time and tackling one element at a time helped me put this complex part of the project together.
