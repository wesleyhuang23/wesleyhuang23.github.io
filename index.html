Fanflix

Made by a Movie Fan for Movie Fans

About

A Movie database website that allows user to search up to 900,000 movie titles. Searching is done in real time using JQuery keyup, along with people and movie theaters. Users can browse movie detailed pages for every searchable movie, included are trailer, posters, backdrops and recommendations for the movie. Users can search for movie theaters in their area and get showtimes at that theater. When logging in clients will be able to create their own movie list and once added can write a review and give a score for the movie.

HTML/CSS | jQuery | AngularJS | NodeJS | Express | SQL | Gulp | OAuth | Heroku
