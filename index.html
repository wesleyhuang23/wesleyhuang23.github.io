Instagram Clone

Sharing Photos comes to the desktop

Features

A clone of the Instagram web app with addition to upload photos and applying filters from computer or phone. Logging in with local auth users can choose to follow people and see their posts on their own feed. Comments and likes are executed in real time, which will be added to a user notifications.

HTML/SCSS | ReactJS | NodeJS | Webpack | Express | MongoDB | Oauth | localAuth | Heroku

Challenges

Setting up Amazon S3 and have photos be uploaded with a link attached to them was complicated in the setup. Re-using components to be modified for a different view was tricky in execution, but careful understanding of how the component was created assisted in the build process.

Fanflix

Made by a Movie Fan for Movie Fans

Features

A Movie database website that allows user to search up to 900,000 movie titles. Searching is done in real time using JQuery keyup, along with people and movie theaters. Users can browse movie detailed pages for every searchable movie, included are trailer, posters, backdrops and recommendations for the movie. Users can search for movie theaters in their area and get showtimes at that theater. When logging in clients will be able to create their own movie list and once added can write a review and give a score for the movie.

HTML/CSS | jQuery | AngularJS | NodeJS | Express | SQL | Gulp | OAuth | Heroku

Challenges

The Challenging part about this project was the back-end. Using OAuth in order to have a user ID be tracked with every action the user does so each user can create their own list of movies. Making sure that the back-end was properly connected. Having some properties be global while others be personal made it more overwhelming, but taking my time and tackling one element at a time helped me put this complex part of the project together.

Flockoli

Taking the Fuss out of what's for lunch

Flockoli is an app built on the FlockOS messaging application. Flockoli allows flock message users to order lunch with a simple /eat command in their chat box. A streamlined process allows users to order lunch while conserving valuable time and effort while at work. Using the /eat command the widget sidebar will apear showing users what choices there are in their local area. Once they select they will be taken to the confirmation page. This was one of my projects at Developer Week Hackathon in San Francisco

HTML/CSS | JQuery | AngularJS | NodeJS | FlockOS
